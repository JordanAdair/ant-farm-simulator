# GitHub Issue Tracker Configuration

Issues for this repository are tracked on GitHub.

- **Platform:** GitHub
- **Repository:** `JordanAdair/ant-farm-simulator`
- **Owner:** `JordanAdair`
- **Repo Name:** `ant-farm-simulator`
- **Base URL:** `https://github.com/JordanAdair/ant-farm-simulator`
- **Issues URL:** `https://github.com/JordanAdair/ant-farm-simulator/issues`

Tools like `to-issues`, `triage`, and `to-prd` should use the `github-mcp-server` or `gh` CLI commands to query or write to this repository.
